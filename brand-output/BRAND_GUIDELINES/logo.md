# Parq AI Brand Guidelines — Logo

---

## Primary Logo: Coordinate Grid — Diamond Variant

The Parq AI logo consists of a **coordinate grid logomark rotated 45° into a diamond orientation** paired with a **dual-typeface wordmark**. The dark background (Parq Navy) is the preferred presentation.

### Logomark
A 3x3 grid formed by intersecting lines, rotated 45 degrees to form a diamond shape — a direct reference to the parking spot mapping technology with added visual dynamism. One node is highlighted in **Parq Teal**, representing the AI's pinpoint detection capability.

### Preferred Background
**Parq Navy (#0A1628)** — The logo is most impactful on the dark navy background, with white grid lines and teal detection node. This is the default for all primary brand touchpoints (website headers, pitch deck covers, business cards).

### Wordmark
- **"Parq"** — General Sans Semibold, Parq Navy
- **"AI"** — JetBrains Mono Medium, Parq Teal, smaller size

The typographic differentiation between "Parq" (geometric sans) and "AI" (monospace) is a core brand element and must always be maintained.

---

## Lockup Variations

### 1. Horizontal (Primary)
Logomark left, wordmark right. "AI" baseline-aligned with "Parq" at smaller size.
- **Min width:** 120px digital, 30mm print
- **Use for:** Website header, email signatures, pitch deck titles

### 2. Stacked
Logomark centered above wordmark. "Parq" first line, "AI" second line, center-aligned.
- **Min height:** 80px digital, 20mm print
- **Use for:** Social media, app splash screens, centered layouts

### 3. Icon Only
Coordinate grid mark without wordmark.
- **Min size:** 24px digital, 8mm print
- **Use for:** Favicons, app icons, watermarks, compact UI

### 4. Wordmark Only
"Parq AI" typeset without the logomark.
- **Min width:** 60px digital, 20mm print
- **Use for:** Inline references, footer credits, very small sizes

---

## Clear Space

Minimum clear space on all sides = the **cap height of the capital "P"** in the wordmark.

For icon-only: clear space = **25% of mark height** on all sides.

When possible, exceed minimum clear space for stronger visual presence.

## Color Applications

### On Light Backgrounds (`#F5F6F8`, `#E8EAF0`)
- Grid lines and "Parq": **Parq Navy `#0A1628`**
- Detection node and "AI": **Parq Teal `#00C2B7`**

### On Dark Backgrounds (`#0A1628`, `#070D1A`)
- Grid lines and "Parq": **Light `#F0F1F4`**
- Detection node and "AI": **Bright Teal `#00D4C8`**

### On Parq Navy Background (`#0A1628`)
- Grid lines and "Parq": **Cloud `#F5F6F8`**
- Detection node and "AI": **Parq Teal `#00C2B7`** (standard — achieves 8.6:1 contrast)

### Monochrome
- **All Navy:** For single-color print, embossing
- **All White:** Reversed on dark/photographic backgrounds
- **All Teal:** Promotional accent applications

## Favicon and App Icons

At very small sizes (below 32px), the grid simplifies to a **2-line cross** (vertical + horizontal) with the teal detection node at the bottom-right intersection.

| Size | Format | Design |
|---|---|---|
| 16x16 | ICO/SVG | Simplified cross + node |
| 32x32 | PNG/SVG | Simplified cross + node |
| 180x180 | PNG | Full grid, Navy background, light lines |
| 192x192 | PNG | Full grid, Navy background |
| 512x512 | PNG | Full grid, Navy background, subtle node glow |
| 1200x630 | PNG | Full horizontal lockup on Navy (OG image) |

## Brand Pattern Extension

The coordinate grid can be extended as a repeating background pattern:
- All lines at **5% opacity** in the foreground color
- **1-3 teal detection nodes** placed at full opacity
- Effect: a live monitoring interface — mostly quiet grid, occasional activity

**Applications:** Hero backgrounds, pitch deck slides, business card reverse, report headers.

---

## Usage Rules

### Do
- Maintain specified clear space
- Use the correct lockup for the context
- Use the correct color version for the background
- Place on solid or near-solid backgrounds
- Give equal or greater prominence when co-branding

### Do Not
- Stretch, compress, rotate, or skew the logo
- Change colors outside defined applications
- Add shadows, glows, or gradients
- Rearrange mark/wordmark relationship
- Set "AI" in the same font as "Parq"
- Make "AI" the same color as "Parq"
- Place on red, orange, or purple backgrounds
- Reproduce below minimum size
- Add text inside the clear space zone
- Use the teal detection node as a standalone element
